# Formatador de Textos Legais

Ferramenta criada utilizando o auxílio de IA para facilitar a formatação de dispositivos legais extraídos de arquivos no formato ".pdf" por intermédio do copia e cola. Visa possibilitar, em especial, a formatação de textos oriundos de PDFs dos Diários Oficiais e de Justiça.

Link: https://y-hobbyist.github.io/formatador-texto-legal/
